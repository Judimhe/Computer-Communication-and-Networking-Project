# Smart City Traffic Control System

A smart city networking project created using Cisco Packet Tracer.  
This project simulates a smart city traffic control network with multiple LANs, WAN connections, wireless access, VoIP communication, and IoT-based network services.

## Project Features
- Multi-zone network design
- Star topology LANs
- WAN serial connections
- Wireless access points
- VoIP communication
- Subnetting and IP addressing
- Routing configuration
- Network testing and verification

## Technologies Used
- Cisco Packet Tracer
- Networking Fundamentals
- Static Routing
- Wireless Networking
- VoIP
- IoT Concepts

## Files Included
- `.pkt` network simulation file
- Project report (PDF)
- Network topology screenshots

## Network Structure
The network contains multiple areas including:
- Customer Service Area
- Public Wi-Fi Zone
- Gaming Zone
- IT Support Department
- External Connectivity Segment
- Administration Office
